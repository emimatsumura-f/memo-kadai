# 課題 ーメモ帳アプリー
友達と一緒に使いたいウォーキング記録アプリを作りました。
高校から仲良しのイツメンと会うと、最近はダイエット・健康・美容の話が多くなり。
年齢に抗おうとがんばってます（笑）海外に住んでる友達も、近くの友達も。
歩こう！とよく話題にわでるのですが、頻繁に集まることも難しいので。
みんなで遊び感覚でウォーキングできたらいいなとおもって作成しました。
イツメンのグループチャットが（BBA）ババアなので、このタイトルです（笑）

## 課題内容
・目標設定ができること
・メンバーが自由に追加できること
・個人のログがわかるようにすること
・進捗状況がわかること
・達成率が可視化できること
・１ヶ月ごとで目標設定したいので、全データがリセットできるボタンをつくること
上記ができるようすることを設定しました。

## 工夫した点・こだわった点
・Googleフォントを利用してフォントを変えてみました
・進捗状況にProgress Barを使用したところ
・リセットボタンをクリックしたときにアラートを発動させるところ

## 難しかった点・出来なかった点
・やりたいことを先に考えて、小さな動作から作成していくようにしました。
・どういう風に関数を作ればいいのか分からず、かなり調べました。調べれば参考にできそうなコードは出てきたのですが、見直したときにコードを読んでもすぐ理解できていないところがあるのが課題です。
じっくり自分でコードを再度確認したいと思います。
・コードを書くことに時間がかかったのでUIにこだわれなかったところです。
みんなが記録することが楽しくなるようなUIにブラッシュアップしたいです。
・目標達成率によって画像が変わる、応援のセリフが表示されるなどやりたかったのですができていないのでブラッシュアップします。
